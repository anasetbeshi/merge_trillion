# إرشادات المساهمة في Coin Hunter 🪙

شكرًا لرغبتك في المساهمة! قبل البدء، يرجى قراءة هذه القواعد بعناية:

## 🟢 مسموح
- **[استخدام الكود](ca://s?q=مسموح_استخدام_الكود)**: نسخ وتعديل الكود مع ذكر المصدر.
- **[إضافة ميزات جديدة](ca://s?q=مسموح_اضافة_ميزات_جديدة)**: مثل تحسين الأداء أو تطوير مستويات اللعبة.
- **[إصلاح الأخطاء](ca://s?q=مسموح_اصلاح_الاخطاء)**: المساعدة في تصحيح المشاكل البرمجية.
- **[إرسال Pull Requests](ca://s?q=مسموح_ارسال_Pull_Requests)**: مع شرح واضح للتغييرات.

## 🔴 ممنوع
- **[نشر بيانات حساسة](ca://s?q=ممنوع_نشر_بيانات_حساسة)**: مثل كلمات المرور أو مفاتيح API.
- **[استخدام محتوى محمي](ca://s?q=ممنوع_استخدام_محتوى_محمي)**: صور أو أصوات بدون إذن أو رخصة.
- **[إضافة كود ضار](ca://s?q=ممنوع_اضافة_كود_ضار)**: أي شيء يضر المستخدمين أو يخرب المشروع.
- **[تغيير الرخصة بدون إذن](ca://s?q=ممنوع_تغيير_الرخصة_بدون_اذن)**: الالتزام بالرخصة الأصلية (MIT).
- **[إساءة استخدام المشروع](ca://s?q=ممنوع_اساءة_استخدام_المشروع)**: لأغراض غير قانونية أو مخالفة.
